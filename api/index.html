James Mayr API

/baseconverter

convert a number from one base to another (full project)

inputs

  • numberString: string
  • oldBase: number
  • newBase: number

output

{"success":bool,"number":string}

/chordanalyzer

get the chord and inversion of a group of notes (full project)

inputs

  • notes: comma-separated string (ex: Eflat,G,Bdoubleflat,Csharp)

output

{"success":bool,"pitches":[numbers],"chord":[strings],"addedNotes":[strings],"rootName":string,"inversion":string,"intervals":[numbers],"chordType":string,"shortType":string,"shortName":string,"longName":string}

/factorfinder

get the factors and fun facts about a number (full project)

inputs

  • number: number

output

{"success":bool,"number":number,"qualities":[strings],"html":HTML string}

/messageencrypter

inputs

  • message: string

/morsemessager

convert between plain text and Morse code (full project)

inputs

  • text: string (optional)
  • code: string (optional)

output

{"success":bool,"output":string}

/random

various random generators (full project)

inputs

  • type: number | string | selection | dice | color (optional)
  • min: number (only for type:"number")
  • max: number (only for type:"number")
  • set: string (only for type:"string")(optional)(ex: "abc123XYZ")
  • size: number (only for type:"string")(optional)
  • list: string (only for type:"selection")(ex: "abc, 123, XYZ")
  • selections: number (only for type:"selection")(optional)
  • groups: number (only for type:"selection")(optional)
  • dice: string (only for type:"selection")(ex: "3d6" or "1d7 * 2 + 13d95")

output

{"success":bool,"output":number|string}

/tiprounder

calculate the gratuity with a rounded total (full project)

inputs

  • amount: number (ex: 12.34)
  • percentage: number (ex: 18)
  • coin: 0.01 | 0.05 | 0.10 | 0.25 | 0.50 | 1.00 (optional)

output

{"success":bool,"tip":number,"total":number}

/unitconverter

convert a quantity from one unit of measurement to another (full project)

inputs

  • quantity: number
  • from: string
  • to: string

output

{"success":bool,"parameters":{"quantity":number,"from":{"prefix":string,"unit":string,"category":string,"system":string},"to":{"prefix":string,"unit":string,"category":string,"system":string}},"output":number,"message":string}

/wordcounter

count the number of characters and words and get the longest word length (full project)

inputs

  • text: string

output

{"success":bool,"characters":number,"words":number,"longest":number}
