homeController is a tool for managing my smart devices.

The buttons on this screen activate a web request to IFTTT, a service that connects triggers (like this webhook) to other APIs. Here, I'm connecting to Philips Hue, Sonos, Roomba, Tuya SmartLife, and Honeywell.

Of course, only if you have the API key... which I can't provide, or you'll mess with my house!

To learn more about my experiments with home automation and API integrations, check out bluejay.
